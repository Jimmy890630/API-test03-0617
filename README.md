# 金屬加工名片系統

本專案為金屬加工產業設計的名片管理系統前端骨架，採用原生 HTML、CSS、JavaScript 開發。

## 專案文件總覽

| 文件 | 說明 |
|------|------|
| [docs/01-requirements.md](docs/01-requirements.md) | 需求規格書：名片欄位、技術規範、開發規則 |
| [docs/02-file-structure.md](docs/02-file-structure.md) | 檔案關係文件：目錄結構、檔案職責與前端載入順序 |
| [docs/03-ui-design.md](docs/03-ui-design.md) | UI / 基礎樣式設計文件：配色、字型、版面、響應式規劃 |

| [docs/05-dev-log.md](docs/05-dev-log.md) | 開發 Log：開發歷程與里程碑 |
| [docs/06-reusable-utils.md](docs/06-reusable-utils.md) | 共用 JS 方法規格：可重複使用方法規劃 |

## 技術堆疊

- HTML5
- CSS3
- JavaScript（ES6+）

## 設計方向

- 簡約專業的工業風格
- 主要配色：深灰藍 `#263238`、鋼藍灰 `#607D8B`、琥珀金 `#FFB300`
- 樣式與互動均以金屬加工產業形象為出發

## 開發狀態

目前僅完成規劃文件，尚未產出任何 HTML、CSS、JavaScript 原始碼。
所有程式碼將在收到「開始開發」指令後才會生成。
