# UI / 基礎樣式設計文件

## 1. 設計方向

- **風格**：簡約、專業、穩重，帶有工業精準感，符合金屬加工產業形象。
- **視覺語彙**：乾淨的線條、足夠的留白、清晰的層級、耐看的配色。
- **模式**：單欄置中名片頁面（Minimal Single Column）。
- **情緒**：專業、穩重、精準。
- **反參考**：太花俏、卡通感、過多裝飾動畫、科技新創鮮豔漸層。

## 2. 色彩系統

本專案採用 OKLCH 色彩空間，並以 impeccable 建議的深靛藍為品牌錨點，搭配琥珀金強調色。

| 角色 | OKLCH | 用途 |
|------|-------|------|
| Primary | `oklch(0.400 0.110 250.0)` | 頁首背景、卡片頂部深色區塊 |
| On Primary | `oklch(1.000 0.000 0)` | 深色背景上的白色文字 |
| Secondary | `oklch(0.550 0.050 250.0)` | 次要資訊、圖示、輔助文字 |
| Accent | `oklch(0.650 0.140 80.0)` | 強調線條、服務標籤、互動回饋 |
| Background | `oklch(1.000 0.000 0)` | 純白頁面背景 |
| Surface | `oklch(1.000 0.000 0)` | 名片卡片背景 |
| Text | `oklch(0.220 0.010 250.0)` | 主要文字，對比度 ≥7:1 |
| Muted | `oklch(0.500 0.020 250.0)` | 次要文字，對比度 ≥3.5:1 |
| Border | `oklch(0.850 0.010 80.0)` | 邊框、分隔線 |

> 配色靈感來自金屬加工的「鋼板、切削、熱處理」意象：深靛藍代表穩重的金屬質感，琥珀金象徵加工時產生的火花與熱度。

## 3. 字體設計

| 用途 | 規劃字型 | 說明 |
|------|---------|------|
| 主要字體 | `Inter` | 透過 Google Fonts 載入，簡約專業 |
| 中文內文 | 系統預設無襯線體（如蘋方、微軟正黑體） | 確保跨平台可讀性 |
| 英文 / 數字 | `Inter`, `Helvetica Neue` | 與中文搭配，數字清晰 |
| 姓名 / 公司名稱 | 字重 Bold（700）/ SemiBold（600） | 強調層級 |
| 聯絡資訊 | 字重 Regular（400）/ Medium（500） | 維持可讀性 |

### 字型大小規劃

| 元素 | 大小 | 字重 |
|------|------|------|
| 頁面標題 | `clamp(1.5rem, 4vw, 1.75rem)` | 600 |
| 名片姓名 | `clamp(1.75rem, 6vw, 2.5rem)` | 700 |
| 公司名稱 | 1.125rem | 600 |
| 職稱 | 0.875rem | 500 |
| 聯絡資訊 | 0.9375rem | 400 |
| 服務項目標籤 | 0.875rem | 600 |
| 輔助文字（統編、傳真） | 0.875rem | 400 |

## 4. 名片卡片規格

| 屬性 | 規格 |
|------|------|
| 寬度 | 最大 560px，單張置中 |
| 內距 | 頂部與主體 2rem，底部 2rem |
| 圓角 | 1rem（16px） |
| 陰影 | `0 12px 32px oklch(0.400 0.020 250.0 / 0.12)` |
| 背景 | `#FFFFFF` |
| 懸停效果 | `translateY(-6px)`、陰影加深，持續 250ms |

## 5. 版面規劃

- **頁首（Header）**：簡潔標題，說明系統名稱。
- **名片內容區**：單張名片置中於畫面。
- **名片卡片內部資訊分區**：
  - 上區：公司名稱 + 職稱
  - 中區：姓名（最重要，放大）
  - 下區：聯絡方式（電話、信箱、LINE）
  - 底部：服務項目標籤、統編 / 傳真

## 6. 響應式規劃

| 螢幕寬度 | 名片排列 |
|---------|---------|
| < 480px | 聯絡資訊單欄排列 |
| ≥ 480px | 聯絡資訊兩欄排列 |
| ≥ 768px | 名片最大寬度 560px，置中 |

## 7. CSS 檔案分工

- `assets/css/main.css`：負責 Reset、CSS 變數、頁面整體 Layout、頁首樣式、響應式 Grid。
- `assets/css/card.css`：負責名片卡片內部的視覺層級、色彩、間距、懸停動畫、標籤樣式。

## 8. 動畫與互動

| 互動 | 效果 |
|------|------|
| 名片卡片 hover | `transform: translateY(-6px)`、陰影加深，持續 250ms |
| 標籤 hover | 背景色變為琥珀金、文字反白 |
| 聯絡連結 hover | 文字顏色變為琥珀金 |
| 焦點狀態 | 2px 琥珀金 outline |

## 9. 無障礙與效能

- 目標符合 WCAG 2.1 AA 對比度標準。
- 尊重 `prefers-reduced-motion`，減少動畫偏好時關閉過渡效果。
- 預先連線 Google Fonts 以加速字體載入。

## 10. 注意事項

- 本文件為規劃文件，實際樣式以 `assets/css/main.css` 與 `assets/css/card.css` 為準。
- 所有 CSS 規則須搭配中文註解說明用途。
- 簡約風格不代表空白，應透過層級、對齊與色彩收斂來達成專業感。
