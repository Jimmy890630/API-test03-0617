# UI / 基礎樣式設計文件

## 1. 設計方向

- **風格**：簡約、專業、工業感，符合金屬加工產業形象。
- **視覺語彙**：乾淨的線條、足夠的留白、清晰的層級、耐看的配色。

## 2. 配色方案

| 角色 | 色碼 | 用途 |
|------|------|------|
| 主色 Deep Slate | `#263238` | 標題、名片主視覺背景、重要文字 |
| 輔色 Steel Blue | `#607D8B` | 次要資訊、邊框、圖示 |
| 強調色 Amber Gold | `#FFB300` | CTA 按鈕、服務項目標籤、重點提示 |
| 背景色 Light Gray | `#F5F5F5` | 頁面背景 |
| 卡片白 White | `#FFFFFF` | 名片卡片背景 |
| 文字色 Dark Gray | `#212121` | 內文文字 |
| 次要文字 Medium Gray | `#757575` | 輔助說明文字 |

> 配色靈感來自金屬加工的「鋼板、切削、熱處理」意象：深灰藍代表穩重的金屬質感，琥珀金象徵加工時產生的火花與熱度。

## 3. 字體設計

| 用途 | 規劃字型 | 說明 |
|------|---------|------|
| 中文內文 | 系統預設無襯線體（如蘋方、微軟正黑體） | 確保跨平台可讀性 |
| 英文 / 數字 | `Roboto`, `Helvetica Neue` | 與中文搭配，數字清晰 |
| 姓名 / 公司名稱 | 字重 Bold（700） | 強調層級 |
| 聯絡資訊 | 字重 Regular（400） | 維持可讀性 |

### 字型大小規劃

| 元素 | 大小 | 字重 |
|------|------|------|
| 頁面標題 | 2rem（32px） | 700 |
| 名片姓名 | 1.5rem（24px） | 700 |
| 公司名稱 | 1.125rem（18px） | 600 |
| 職稱 / 服務項目 | 1rem（16px） | 400 / 500 |
| 聯絡資訊 | 0.875rem（14px） | 400 |
| 輔助文字（統編、傳真） | 0.8125rem（13px） | 400 |

## 4. 名片卡片規格

| 屬性 | 規格 |
|------|------|
| 寬度 | 最少 320px，最大依版面設定（響應式） |
| 內距 | 1.5rem（24px） |
| 圓角 | 0.75rem（12px） |
| 陰影 | `0 2px 8px rgba(38, 50, 56, 0.08)` |
| 背景 | `#FFFFFF` |
| 懸停效果 | 輕微上浮與陰影加深 |

## 5. 版面規劃

- **頁首（Header）**：簡潔標題，說明系統名稱。
- **名片列表區**：使用 CSS Grid / Flexbox 排列名片卡片。
- **名片卡片內部資訊分區**：
  - 上區：公司名稱 + 職稱
  - 中區：姓名（最重要，放大）
  - 下區：聯絡方式（電話、信箱、LINE）
  - 底部：服務項目標籤、統編 / 傳真

## 6. 響應式規劃

| 螢幕寬度 | 名片排列 |
|---------|---------|
| ≥ 1024px | 每行 3 張 |
| 768px ~ 1023px | 每行 2 張 |
| < 768px | 每行 1 張 |

## 7. CSS 檔案分工

- `assets/css/main.css`：負責 Reset、CSS 變數、頁面整體 Layout、頁首樣式、響應式 Grid。
- `assets/css/card.css`：負責名片卡片內部的視覺層級、色彩、間距、懸停動畫、標籤樣式。

## 8. 動畫與互動

| 互動 | 效果 |
|------|------|
| 名片卡片 hover | `transform: translateY(-4px)`、陰影加深，持續 0.2s |
| 標籤 hover | 背景色變深、文字反白 |
| 按鈕點擊 | 輕微 scale(0.98) 回饋 |

## 9. 注意事項

- 本文件為規劃文件，尚未產生實際 CSS 檔案。
- 開始開發後，所有 CSS 規則須搭配中文註解說明用途。
- 簡約風格不代表空白，應透過層級、對齊與色彩收斂來達成專業感。
