# DESIGN.md

## 專案：金屬加工名片系統

本文件定義名片頁面的視覺系統，包含色彩、字體、版面與元件規範。

## 設計方向

- **風格**：簡約、專業、穩重，帶有工業精準感
- **模式**：單欄置中名片頁面（Minimal Single Column）
- **情緒**：專業、穩重、精準
- **反參考**：太花俏、卡通感、過多裝飾動畫、科技新創鮮豔漸層

## 色彩系統

| 角色 | 色碼 | 用途 |
|------|------|------|
| Primary | `#1C1917` | 頁首背景、主要標題、名片深色區塊 |
| On Primary | `#FFFFFF` | 深色背景上的文字 |
| Secondary | `#44403C` | 次要資訊、圖示、輔助文字 |
| Accent | `#A16207` | 強調線條、標籤、互動回饋 |
| Background | `#FAFAF9` | 頁面背景 |
| Surface | `#FFFFFF` | 名片卡片背景 |
| Text | `#0C0A09` | 主要文字 |
| Muted | `#78716C` | 次要文字、說明 |
| Border | `#D6D3D1` | 邊框、分隔線 |

## 字體系統

- **標題字體**：`Inter`（500–700）
- **內文字體**：`Inter`（400–500）
- **中文後備**：`PingFang TC`、`Microsoft JhengHei`
- **載入方式**：Google Fonts CSS import

## 間距系統

| Token | 數值 |
|-------|------|
| xs | 0.25rem |
| sm | 0.5rem |
| md | 1rem |
| lg | 1.5rem |
| xl | 2rem |
| xxl | 3rem |

## 名片版面

- 單張名片置中於畫面
- 最大寬度：560px
- 卡片結構：
  1. 頂部深色區塊：公司名稱、職稱
  2. 主體：姓名、聯絡資訊
  3. 底部：服務項目標籤、統編與傳真

## 互動規範

- 電話與信箱為可點擊連結
- 卡片懸停時輕微上浮並加深陰影
- 所有互動元素有 150–300ms 的平滑過渡
- 尊重 `prefers-reduced-motion`

## 響應式斷點

- 375px：手機
- 768px：平板
- 1024px：桌機
- 1440px：大螢幕
